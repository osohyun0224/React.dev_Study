- 먼저, React란 사용자 인터페이스를 구축하기 위한 선언적이고 효율적이며 유연한 자바스크립트 라이브러리 입니다.
- React를 써서 프로젝트 개발을 하면서, 기능 구현에 치중했던 지난 날을 반성하며 방학에라도 공식문서를 공부하고 싶어 시작합니다!
- 전체적인 React의 이해와 본질을 최대한 많은 내용을 내 것으로 만들자!
